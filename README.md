# react-native-medcheck-sdk

Wrapper for medchecksdk

## Installation

```sh
npm install react-native-medcheck-sdk
```

## Usage

```js
import MedcheckSdk from "react-native-medcheck-sdk";

// ...

const result = await MedcheckSdk.multiply(3, 7);
```
